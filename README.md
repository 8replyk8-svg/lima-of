# LIMA - OF · Registros Clientes Digitales

## Archivos
- `registros-clientes.html` — la página completa (ábrela con Live Server en VS Code, o súbela a GitHub Pages).
- `apps-script-registros.gs` — pégalo en Extensiones > Apps Script del Google Sheet de prueba.

## Ya configurado
- Link del Apps Script: fijo en el código.
- Token de acceso: fijo en el código (mismo en ambos archivos).
- PIN de entrada a la página: 2025.
- Listas de Clase / Uso / Tipo Pago editables desde el ícono ⚙️ dentro de la página.

## Para desplegar en GitHub Pages
1. Sube `registros-clientes.html` a tu repo.
2. Settings → Pages → Branch: main → Save.
3. Ábrelo desde `https://tuusuario.github.io/turepo/registros-clientes.html`.

## Si cambias el código del Apps Script
Después de editar y guardar, hay que republicar:
Implementar → Gestionar implementaciones → ✏️ → Nueva versión → Implementar.
(Si no haces esto, la URL sigue sirviendo la versión vieja.)
